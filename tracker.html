Stock Portfolio Tracker | Python, yfinance, Pandas, Matplotlib, Flask

____________

A command-line tool that fetches real-time stock prices, calculates portfolio value and profit/loss, and generates a pie chart of asset allocation. Demonstrates API integration, data processing, and visualization.

Key Features

  • Fetches live stock prices using the yFinance API (no API key required)
  • Reads user holdings from a CSV file (symbol, shares, purchase price)
  • Calculates total portfolio value, daily change, and unrealized gain/loss
  • Displays a formatted report in the terminal
  • Generates a pie chart with Matplotlib showing allocation with live prices per share
  • Handles errors (invalid symbols, missing files, API failures)
